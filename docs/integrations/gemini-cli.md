## Prerequisites

Same as [claude-code.md](claude-code.md). Plus Gemini CLI installed: <https://github.com/google-gemini/gemini-cli>.

## 1. Install the bridge

See [claude-code.md §1](claude-code.md#1-install-the-bridge).

## 2. Fetch the connect snippet

```bash
curl -s "http://localhost:3000/api/v1/connect/gemini-cli?token=$HELMDECK_TOKEN" | jq .
```

## 3. Configure Gemini CLI

Config path: `~/.gemini/settings.json`. Merge:

```json
{
  "mcpServers": {
    "helmdeck": {
      "command": "helmdeck-mcp",
      "env": {
        "HELMDECK_URL": "http://localhost:3000",
        "HELMDECK_TOKEN": "<your-jwt>"
      }
    }
  }
}
```

Restart `gemini` and run `/mcp` to confirm the connection.
